12) . 붙이면 class 참조

```html
 <style>
        h1 {
            color: red;
        }
        h2 {
            color: blue;
        }
        #green {
            color: green; 
        }
        .yellow {
            color: orange;
        }
    </style>
</head>
<body>
    <!-- 이것은 주석입니다
         여러줄 가능합니다 -->
    <h1>이것은 h1 태그입니다.</h1>
    <h2>이것은 h2</h2>
    <h3>이것은 h3</h3>
    <h4 id="green">이것은 h4</h4>
    <h5 class="yellow">이것은 h5 태그입니다.</h5>
    <h6 class="yellow">이것은 h6 입니다.</h6>

    <p>이것은 p 태그입니다. <br>
        p 태그라고 합니다. <br>
        p 태그라고 하네요. <br>
    </p>

    <ul>
        <li>리스트 1번째</li>
        <li>리스트 2번째</li>
        <li>리스트 3번째</li>
    </ul>

    <ol>
        <li>리스트 1번째</li>
        <li>리스트 2번째</li>
        <li>리스트 3번째</li>
    </ol>

    <div>
       <b>여기는 div 입니다 굵은 글씨체.</b> 
       <i>기울임 글꼴입니다</i>
    </div>
</body>
```



* 우선순위 : 인라인 > id > class > tag
